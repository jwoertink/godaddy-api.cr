# Godaddy::API

A simple API wrapper for [Godaddy](https://www.godaddy.com/)

## Installation

Add this to your application's `shard.yml`:

```yaml
dependencies:
  godaddy-api:
    github: jwoertink/godaddy-api.cr
    branch: master
```

## Usage

